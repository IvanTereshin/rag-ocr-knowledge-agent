# Web Demo

React + Vite frontend for the portfolio version of the RAG OCR Knowledge Agent.

This first iteration uses mocked data only. It is built for client demonstrations and can be published as a static frontend.

## Commands

Run from the repository root:

```bash
npm run dev
npm run build
npm run preview
```

Or run directly inside `apps/web`:

```bash
npm install
npm run dev
```

## Current Scope

- bilingual UI: Russian and English;
- mocked document processing states;
- mocked OCR/RAG pipeline;
- mocked answer with citations;
- `portfolio-top` as the active profile;
- `local-self-host` shown as a planned second stage.
